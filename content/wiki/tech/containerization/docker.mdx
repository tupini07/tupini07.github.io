Docker is the main containerization technology

## Tools

- [!!Portainer](WID:20200715221943957) - a tool to manage docker installations
    via web interface. Supports both local and remote installations.
- [!!Traefik](WID:20200716134258059) - a *super reverse proxy* that routes
    requests to linked services (be them exposed by Docker or not).
- [!!Watchtower](WID:20200716134548536) - A process for automating Docker
  container base image updates
